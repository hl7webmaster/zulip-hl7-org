Rob McClure (May 20 2019 at 00:50):

In the USA the CDC Race and Ethnicity code system, which is used in many standards as THE source for concepts designated as Ethnicity, considers only hispanic concepts to be a "type of ethnicity." All of the many other cultural distinctions are considered "Race" types. You can see this by looking at the code system, https://vsac.nlm.nih.gov/context/cs/codesystem/CDCREC/version/1.2/code/1000-9/info. Equally perplexing, you will not find any mention of either of the two Jewish hereditary groups you mention anywhere in this code system. What to do? I'd say that given that in the US CDC says only Hispanic and Non-hispanic are ethnicities while everything else is a "race." So if you follow the logic there, you should add the two you need to the Race component. But if you want to consider them ethnicities I certainly wouldn't argue. Since I'm not sure where others have traditionally put this for V2 messages, you might wait to see how others respond.
