Lloyd McKenzie (Apr 05 2019 at 14:08):

There aren't separate schemas for different CDA implementation guides. All CDA implementations use the same schema. However, some CDA implementation guides are accompanied by schematrons that impose some, sometimes even most of the rules defined in the IG. I'm not sure where those would be located for this IG though. @Lisa Nelson?

Lisa Nelson (Nov 07 2019 at 18:37):

David, the CDA R2.0 Schema is stored in GForge: https://gforge.hl7.org/gf/project/strucdoc/frs/ and the C-CDA R2.1 IG includes in the publication package for the Guide. The schematron is specific for validating conformances defined in the templates while the Schema is specific for the defining the base CDA standard.
