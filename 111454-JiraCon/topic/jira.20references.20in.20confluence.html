Joshua Procious (Dec 09 2019 at 16:57):

The easiest way to insert a Jira Link while editing a page in Confluence is to use the "insert Macro" shortcut (left curly bracket) { and then type "Jira". Hit enter and the "Insert Jira Issue/Filter will pop up. From here you can search using jql or predefined filter, you can create new issues, and you can select from ones you've recently viewed (usually the quickest in my experience).
