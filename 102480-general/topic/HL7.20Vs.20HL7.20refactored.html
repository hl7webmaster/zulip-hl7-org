Mo Aamer (Mar 02 2020 at 10:44):

excuse my question if it is dumb, is there any difference between "HL7" and "HL7 refactored", if so please explain. thanks in advance.

Craig Newman (Mar 02 2020 at 14:19):

If by "HL7 refactored" you mean this site: http://www.hl7.eu/refactored/hl7.html, then there is a difference. That site is a test for a future move to web based publishing of the v2 base standard. That site is not an official source of any v2 standards. The official v2 standards are downloadable from the HL7 master product grid.

Frank Oemig (Mar 05 2020 at 08:26):

Correct. The refactored site provides the contents of v2.8.2 but in a new way that should be easier to understand. Feedback and errors are welcome.
